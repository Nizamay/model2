# DoorCalc Pro

Калькулятор стоимости межкомнатных дверей — веб-приложение (один файл, HTML+CSS+JS, без сборки и зависимостей).

## Запуск

Откройте `index.html` в любом браузере (двойным кликом или через любой веб-хостинг — файл полностью самодостаточный).

## Статус

- v0.2 — расчёт дверного полотна: модель, материал (ПВХ/эмаль), цвет с автоопределением категории наценки, стандарт/нестандарт размера (+30%, округление до 10 ₽), опции, скидка.
- Базовые цены моделей и цена стёкол — временные заглушки (см. комментарий в начале `<script>` внутри `index.html`), реальных цен не было в исходных данных.
